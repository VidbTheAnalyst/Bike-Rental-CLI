A simple Python CLI project that simulates renting and returning bikes with **time-based billing**.  
Built using basic **OOP concepts**, easy to understand and beginner-friendly.

## Features
- Rent bikes with customer name & quantity  
- Return bikes with automatic duration-based billing  
- Hourly charges (rounded up)  
- Simple stock management  
- Clean command-line interface  

## How It Works
1. Choose **Rent a Bike**  
2. Enter name & quantity  
3. When returning, the system calculates total time and price  
4. View current rental status anytime
